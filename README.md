# C++ Game Engine
[Work in Progress]

Small 2D game engine built using C++, with some additional libraries for input handling and rendering.
Using Entity-Component-System (ECS) pattern for core engine functionality


## Using:
- C++ 17
- SFML 3.0.0
- imgui v1.91.8
- imgui-sfml 2.6.1
